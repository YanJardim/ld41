Combine 2 Incompatible Genres

card rpg rogue like {

    Monta seu deck com 3 cartas de ação, 2 equipaveis
    Jogo por turno
    Sempre o jogador começa
    Sempre que jogar uma carta ela fica 1 turno sem poder ser jogada, a n ser que a carta diga o contrario
    
    Status {
        Health
        Strength
        Defense
        Agility
    }

    Cartas {
        Ação {
            Upar de level - Aumenta todos seus status, só pode ser usada 1x por partida
            Atacar com arma primaria
            Atacar com arma secundaria
            Defender - não toma dano essa rodada
        }
        Equipaveis {
            Espada longa
            Escudo - da bonus na defesa e chance de esquivar golpe
            Arco e flecha - precisa ter algo de diferente da espada
        }
        
    }

}
